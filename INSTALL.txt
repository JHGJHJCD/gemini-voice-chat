שיחה קולית עם Gemini - הוראות התקנה 🎙️
==========================================

התקנה (פעם אחת):

1. לחץ פעמיים על GeminiVoiceChat-Setup.exe

2. ⚠️ אם Windows מציג אזהרה כחולה ("Windows protected your PC"):
   - לחץ על "More info" (מידע נוסף)
   - לחץ על "Run anyway" (הפעל בכל זאת)
   זה תקין! האזהרה מופיעה כי התוכנה לא נרכשה עם תעודה מסחרית בתשלום.
   התוכנה בטוחה לחלוטין.

3. במסך ההתקנה, מומלץ לסמן:
   ☑ "התקן תעודת אבטחה" - מסיר אזהרות עתידיות
   ☑ "צור קיצור דרך בשולחן העבודה"

4. לחץ Install (התקן) וסיים.

----------------------------------------

הפעלה ראשונה:

1. פתח את התוכנה (מתפריט Start או משולחן העבודה)

2. תתבקש מפתח API חינמי של Gemini:
   - היכנס ל: https://aistudio.google.com/app/apikey
   - צור מפתח (Create API Key) והעתק
   - הדבק בחלון ולחץ "התחל"

3. לחץ "התחל שיחה" ודבר! 🎉

----------------------------------------

טיפים:
- מקש רווח = התחל/עצור שיחה
- אפשר להפריע ל-Gemini באמצע - פשוט תתחיל לדבר
- ⚙ = בחירת קול, מיקרופון, אוזניות, צבעים
- 👤 = הגדרת אישיות (מורה, מתרגם...)
- 🖥️/📷 = שיתוף מסך או מצלמה
- 💾 = שמירת השיחה לקובץ

דרישות: Windows 10/11, אינטרנט, מיקרופון ורמקול.

להסרה: "הוסף או הסר תוכניות" בהגדרות Windows.
