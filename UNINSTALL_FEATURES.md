# 🗑️ תכונות הסרה — v1.5

התוכנה v1.5 כוללת שלוש דרכים להסרה **חכמות**:

---

## **1️⃣ הסרה אוטומטית בזמן התקנה** ⭐ (חדש!)

### **מה קורה:**
כשאתה מתקין את v1.5, ה-installer **מוחק אוטומטית** כל גרסה ישנה:

```
GeminiVoiceChat-Setup.exe (v1.5)
    ↓
בדיקה: האם קיימת גרסה ישנה?
    ↓
אם כן: הרץ את ה-uninstaller ישן
    ↓
מחק את התיקייה הישנה
    ↓
התקן v1.5 חדש
```

### **יתרון:**
✓ **אתה לא צריך לעשות שום דבר**  
✓ Installer עושה את הניקוי בעצמו  
✓ אפילו לא צריך לכבות את התוכנה הישנה

### **איך זה עובד:**
```pascal
// בתוך installer.iss:
if FileExists(unins000.exe) then
    Exec(unins000.exe, '/SILENT')  { הרץ בשקט }
    Sleep(2000)                      { חכה לסיום }
    DelTree(olddir, True, True, True) { מחק שאריות }
```

## **2️⃣ כפתור הסרה בתוך התוכנה** (חדש!)

### **איפה:**
```
⚙️ כפתור הגדרות (בימין העליון)
    ↓
דיאלוג הגדרות נפתח
    ↓
גללול לתחתית
    ↓
🗑️ כפתור אדום: "הסר את התוכנה מהמחשב"
```

### **איך להשתמש:**

1. **לחץ על ⚙️ (הגדרות)** בחלון הראשי

2. **גלול למטה** בדיאלוג ההגדרות

3. **לחץ על כפתור אדום:**
   ```
   🗑️ הסר את התוכנה מהמחשב
   ```

4. **אזהרה:**
   ```
   ⚠️ האם תרצה להסיר?
   
   פעולה זו:
   ✓ מוחקת את התוכנה וקבצי הליבה
   ⚠️ משמרת את הגדרותיך ומסמכיך
   
   [לא]  [כן]
   ```

5. **לחץ "כן":**
   - ה-uninstaller עובד בשקט
   - התוכנה סוגרת את עצמה
   - סיום!

### **מה מתחוק:**
```
✓ התוכנה עצמה
✓ קבצי ההתקנה
✓ Registry (רישום Windows)

⚠️ משמרת:
✓ api_key.txt (המפתח שלך)
✓ settings.json (הגדרות)
✓ memory.json (שיחות שנשמרו)
✓ knowledge.json (מסמכים)
```
