## 🤖 בדיקות אוטומטיות

### הרצה מהירה
```bash
# התקן pytest אם אין
pip install pytest

# הרץ את כל הבדיקות
pytest test_suite.py -v

# או בדיקה ספציפית
pytest test_suite.py::test_config_defaults -v
```

### מה מכוסה

| מודול | בדיקות | סטטוס |
|-------|--------|-------|
| `config.py` | 3 בדיקות: save/load, corrupted JSON, defaults | ✅ |
| `knowledge.py` | 5 בדיקות: memory, kb, add/remove, limits | ✅ |
| `documents.py` | 5 בדיקות: extract, truncate, errors, encoding | ✅ |
| `computer_tools.py` | 6 בדיקות: all 4 functions + edge cases | ✅ |
| `wakeword.py` | 3 בדיקות: init, start/stop, keywords | ✅ |
| `voice_engine.py` | בדיקות אינטגרציה (דורש API) | 🔶 |

**סה"כ:** 22 בדיקות יחידה (ללא רשת) + 1 אינטגרציה

### הרצה עם API
אם יש לך Gemini API key תקין:
```bash
export GEMINI_API_KEY="your-key-here"
pytest test_suite.py -v
```

## ⚠️ בדיקות יציבות

**זה המקום שבו התקיעו האגים הקשים!** 

### 1️⃣ Start/Stop × 50
**מטרה:** וודא שזרמים, threads, websockets נסגרים נקי.

```bash
# בתיקיית הפרויקט:
python -c "
import time
import psutil
import subprocess
import sys

proc = psutil.Process()
baseline_handles = proc.num_handles() if sys.platform == 'win32' else 0

for i in range(50):
    print(f'Cycle {i+1}...')
    # (כאן הייתה הרצת האפליקציה וסגירתה)
    time.sleep(0.5)

final_handles = proc.num_handles() if sys.platform == 'win32' else 0
print(f'Handles: {baseline_handles} → {final_handles}')
if final_handles > baseline_handles + 10:
    print('[FAIL] זליגת handles!')
else:
    print('[OK] נקי')
"
```

### 2️⃣ שיחה ארוכה (30+ דקות)
- הפעל את האפליקציה
- תדבר עם Gemini בשיחה רציפה ל-30 דקות או יותר
- צפה ב-Task Manager:
  - **זיכרון (Memory)** — לא אמור לטפס
  - **Handles** — לא אמור לגדול
  - **CPU** — צריך לחזור ל-0-5% בשקט

**סימני אזהרה:**
- 🔴 זיכרון גדל באופן לינארי → זליגה
- 🔴 Handles לא חוזרים → תיקייה נשאירה פתוחה
- 🔴 GPU Memory עולה → לא משחררים frames

### 3️⃣ החלפת התקנים בזמן
- התחל שיחה עם **רמקולים**
- באמצע: שנה ל**אוזניות** או ה**סר**
- בדוק:
  - ✅ ממשיך לשמוע את הקול
  - ✅ אין ניתוקים/שגיאות בתמלול

### 4️⃣ וידאו on/off × 20
- תדליק ותכבה וידאו 20 פעמים
- בדוק:
  - ✅ **mss** (screen) משתחררת
  - ✅ **camera** משתחררת
  - ✅ אין reset מצלמה מיותר

### 5️⃣ Wake Word מפעיל שעה
- הדלק את "מילת הפעלה" בהגדרות
- תן ל-listener לרוץ שעה בלי להפריע
- בדוק:
  - ✅ הlistener לא קורס
  - ✅ אין זליגת זיכרון
  - ✅ microphone device אינו תקוע

### 6️⃣ מעבר מגש ↔ חלון × 10
- לחץ על אייקון מגש כדי להחביא
- לחץ שוב כדי להציג
- עשה זאת 10 פעמים
- בדוק:
  - ✅ **תמיד חלון אחד** (לא כפול)
  - ✅ אין icons כפולים במגש

### B. דיכוי הד

|--------|------|
| Gemini מדבר בווליום **נמוך** | לא שומע את עצמו |
| Gemini מדבר בווליום **בינוני** | לא שומע את עצמו |

### C. התפרצות חכמה (Barge-in)
- [ ] כשGemini מדבר, אתה **מדבר בקול רם** → הוא עוצר ומקשיב
- [ ] דיבור **שקט** בזמן שהוא מדבר → הוא לא מפסיק (נחמד!)
- [ ] **כוונון הסף** (`BARGE_IN_LEVEL = 0.22`) — בדוק אם זה טוב לסביבתך

### D. סנכרון תמלול
- [ ] התמלול מופיע בזמן אמת
- [ ] אין עיכוב ניכר
- [ ] משפטים סיימים כשGemini סיים לדבר

### E. עברית
- [ ] הגייה נכונה (אין "אות" במקום "אות")
- [ ] ניקוד — אם יש (בד"כ לא)
- [ ] RTL display — כותר וטקסט מימין לשמאל

### F. אנימציה
- [ ] **כדור קולי** זז בהתאם לעוצמת קול של המיקרופון
- [ ] רגיש אבל לא רועד

## 📦 בדיקות בנייה והפצה

### A. התקנה במחשב נקי (קריטי!)

**יצר VM או לוותר מחשב זמני:**

1. ✅ **בלי Python מותקן** — exe עצמאי
2. ✅ **בלי VC++ Redistributable** — התקנה מייד
3. ✅ **Windows 10 ו-11** — שניהם

**קריטי — בדוק זאת!**

```
מורידים GeminiVoiceChat-Setup.exe
↓
מריצים הinstallerים
↓
בחרים "התקן תעודה" (אפשרי)
↓
מופיע אשף מפתח API בהפעלה ראשונה
↓
מכניסים מפתח Google
↓
לוחצים על כדור
↓
["Listening..."] מופיע
↓ מדברים
↓
בדוק: ✅ Gemini מענה
```

### B. תפריט Start
```
הגדרות Windows → אפליקציות → יישומים מותקנים
↓
חיפוש: "שיחה קולית"
↓
צריך: Install, Uninstall, תאריך התקנה
```

### C. הסרה
```
Windows → Uninstalleren → בחר בתוכנה
↓
"התוכנה הוסרה בהצלחה"
↓
בדוק: ✅ לא נשאר תיקיה
↓
בדוק: ✅ האייקון נעלם מ-Desktop (אם בחרו)
```

### D. עדכון אוטומטי
```
בהגדרות → בדוק עדכונים
↓
צריך: "גרסה חדשה זמינה" (אם יש)
↓
תקבל קישור להורדה
```
